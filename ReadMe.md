# Asp .Net MVC Application Labor Book
# <sub>The following technologies are used in this application:<sub>
1. ASP .Net MVC Framework v4.7.2
2. Entity Framework v6.4.0
3. HTML/CSS/JavaScript
4. JQuery v3.4.1
5. BootStrap v4.4.1
